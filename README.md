# My New Repository

## Install env with python -m venv

Do:
```
python -m venv env
source env/bin/activate
```

## Install requirements.txt

Do:
```
pip install -r requirements.txt
```

## create requirements.txt

Do:
```
pip freeze > requirements.txt
```
